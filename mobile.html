Spriteok

Mi is az a sprite?

A Sprite osztály a Pygame nevű játékfejlesztési könyvtár része, és segítségével könnyen kezelheted és szervezheted a játékban megjelenő objektumokat.
A Sprite osztály az összetett játékokban rendkívül hasznos, mivel lehetővé teszi a játékobjektumok egységes kezelését és csoportokba rendezését.
A Sprite osztályból származtatott saját osztályokat (pl. Player, Enemy, Bullet stb.) hozhatsz létre, és ezeket könnyen kezelheted a játéklogikában. A Pygame Sprite rendszere optimalizált, így hatékonyan lehet vele dolgozni.
Ez a kód egy Block nevű osztályt definiál Pythonban, amely a Pygame könyvtár használatára épül.
A Block osztály létrehoz egy sprite-ot (grafikus objektumot) egy játékterületen.
Néhány fontos dolog a kódban:

A __init__ függvény inicializálja az osztályt, és három bemeneti paramétert vár: color (szín), width (szélesség), és height (magasság)

A pygame.sprite.Sprite.init(self) hívja meg az ősosztály (Sprite) inicializáló függvényét

Az image változó egy kétdimenziós felületet hoz létre a megadott színű és méretű téglalap alakú blokkal

A rect változó tárolja a blokkhoz tartozó téglalap alakú területet

Ez a kód részlet lényegében egy egyszerű blokk létrehozását valósítja meg a Pygame sprite rendszerén keresztül. A létrehozott blokkot később hozzá lehet adni egy játékterülethez vagy hasonló felhasználásra 


class Block(pygame.sprite.Sprite):
      def __init__(self, color, width, height):
          pygame.sprite.Sprite.__init__(self)
          self.image = pygame.Surface([width, height])
          self.image.fill(color)
          self.rect = self.image.get_rect()

Példa a Sprite-ok használatára

Ez a kódrészlet egy egyszerű Pygame alkalmazást mutat be, ahol egy Sprite objektumot hozunk létre és mozgatunk véletlenszerűen.
A MySprite osztály a "pygame.sprite.Sprite" osztályból származik. Ez azt jelenti, hogy a MySprite osztály egy típusa a Pygame által biztosított Sprite osztály leszármazottja.
Az osztályban az __init__ metódusban inicializáljuk a Sprite objektumot.
A super().__init__() segítségével hívjuk meg a szülőosztály konstruktorát, ami beállítja az alapértelmezett Sprite tulajdonságokat.
A self.image változó egy pygame.Surface objektum, amely egy kép felületét reprezentálja. Ebben az esetben egy 50x50 pixel méretű piros négyzetet hozunk létre (self.image.fill((255, 0, 0))).
A self.rect változó egy olyan téglalapot (rect) reprezentál, amely a Sprite pozícióját és méretét tartalmazza. Ezt az értéket a self.image.get_rect() hívással kapjuk meg, majd beállítjuk a pozícióját a képernyő közepére (self.rect.center = (screen_width // 2, screen_height // 2)).
Az update metódusban frissítjük a Sprite objektumot. Ebben a példában a rect.x és rect.y értékeit véletlenszerűen változtatjuk meg, így a Sprite objektum véletlenszerűen mozogni fog.
Létrehozunk egy pygame.sprite.Group objektumot, ami egy gyűjtőhely a Sprite objektumok számára.
Létrehozunk egy MySprite objektumot (my_sprite), majd hozzáadjuk a Sprite csoportunkhoz a all_sprites.add(my_sprite) sorral.
Ez az egyszerű példa egy ablakban megjelenő piros négyzetet mutat be, amely véletlenszerűen mozog. A Pygame Group osztály segítségével könnyen kezelhetjük és frissíthetjük az összes Sprite objektumot egyetlen lépésben.


    class MySprite(pygame.sprite.Sprite):
         def __init__(self):
            super().__init__()
            self.image = pygame.Surface((50, 50))
            self.image.fill((255, 0, 0))

            self.rect = self.image.get_rect()
            self.rect.center = (screen_width // 2, screen_height // 2)

         def update(self):
            self.rect.x += random.randint(-5, 5)
            self.rect.y += random.randint(-5, 5)

            all_sprites = pygame.sprite.Group()
            my_sprite = MySprite()
            all_sprites.add(my_sprite)

PlayerSprite osztály definiálása: Az osztály a pygame.sprite.Sprite osztályból származik, ami azt jelenti, hogy ez egy Pygame Sprite objektum.

A __init__ metódusban inicializáljuk a játékobjektumot.
A self.image változó egy 50x50 pixel méretű kék négyzetet reprezentáló felület (pygame.Surface). Ez lesz a karakterünk megjelenése.

A self.rect változó egy olyan téglalapot reprezentál, amely a karakter pozícióját és méretét tárolja. A pozíciót a képernyő közepén helyezzük el (self.rect.center). self.y_speed változó tárolja a karakter Y irányú sebességét, amely az ugrást fogja irányítani.

Az update metódus:
A metódus felelős a játékobjektum frissítéséért. Ebben az esetben a karakter felfelé mozog a self.y_speed sebesség értékének megfelelően. Szimuláljuk a gravitációt: ha a karakter a levegőben van (self.rect.y kisebb, mint a képernyő magassága mínusz a karakter magassága), akkor a gravitáció hatására gyorsul felfelé, különben a karakter a talajon van, és a Y sebessége nullára áll be, tartva a karaktert a talajon.

A jump metódus:
A metódus beállítja a karakter ugrását. Az ugrás csak akkor lehetséges, ha a karakter a talajon van (azaz self.rect.y egyenlő a képernyő magassága mínusz a karakter magassága).

all_sprites Sprite csoport létrehozása:
Egy pygame.sprite.Group objektumot hozunk létre, amely egy gyűjtőhely a Sprite objektumokhoz.

Egy PlayerSprite objektum létrehozása és a csoportba adása:
Egy player nevű PlayerSprite objektumot hozunk létre.
Hozzáadjuk a player objektumot a all_sprites csoportunkhoz a all_sprites.add(player) sorral.
Ez a kódrészlet egy karaktert reprezentál, amely képes ugrani és a képernyő tetején visszaesni a talajra, szimulálva a gravitációt. A karakter mozgatásához a SPACE billentyűt lehet lenyomni.


    class PlayerSprite(pygame.sprite.Sprite):
        def __init__(self):
            super().__init__()
            self.image = pygame.Surface((50, 50))
            self.image.fill((0, 128, 255))
            self.rect = self.image.get_rect()
            self.rect.center = (screen_width // 2, screen_height // 2)
            self.y_speed = 0

        def update(self):
             self.rect.y += self.y_speed
             if self.rect.y < screen_height - 50:
                self.y_speed += 1
             else:
                self.y_speed = 0
                self.rect.y = screen_height - 50

        def jump(self):
             if self.rect.y == screen_height - 50:
                self.y_speed = -15

    all_sprites = pygame.sprite.Group()

    player = PlayerSprite()
    all_sprites.add(player)
